# Markdown 文档知识库

一个零依赖的本地 Markdown 查看、编辑、搜索和知识图谱页面。

## 启动

```powershell
node server.js
```

打开 `http://localhost:4173`。

## 使用

- 把 Markdown 文件放到 `docs/` 目录。
- 左侧按文件夹显示目录。
- 搜索框支持全局关键字检索。
- “修改”模式可保存当前 `.md` 文件。
- “图谱”会展示文档之间的链接、标签和关键词关联。
- 在“修改”模式下可以直接粘贴截图，图片会压缩后保存到 `source/`，并自动插入 Markdown 图片路径。
- 左侧支持新建文件夹和在当前文件夹中新建 Markdown 文件。
- 编辑工具栏支持加粗、下划线、删除线、高亮、文字颜色、背景色、字号和分割线。
- 编辑模式支持 `Ctrl+Z` 撤销和 `Ctrl+Y` / `Ctrl+Shift+Z` 重做。
- 支持删除当前选中的 Markdown 文件或文件夹。
- 删除操作使用应用内确认弹窗；打开文件时默认删除当前文件，只有明确点击文件夹后才删除文件夹。
- 支持拖动 Markdown 文件到任意文件夹，也可以拖到目录空白处回到 `docs/` 根目录。
- 支持拖动文件夹到其他文件夹，调整目录层级。
- 知识图谱会显示共享关键词节点，文档会通过关键词节点形成关联。
- 独立软件打包目标见 `STANDALONE.md`，后续应使用 WebView2/原生壳，避免 Electron。
