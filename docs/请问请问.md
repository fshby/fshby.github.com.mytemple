# 🌟 MyTemple Knowledge —— 下一代个人知识管理利器

---

## 🚀 产品亮点

### 📚 智能文档管理
- **多工作区支持**：自由导入本地文件夹，构建专属知识仓库
- **文件树导航**：层级分明的文件组织，快速定位目标文档
- **智能搜索**：全文检索，毫秒级定位关键信息

### ✍️ 沉浸式编辑体验
- **三模式切换**：阅读模式、编辑模式、知识图谱模式无缝切换
- **实时预览**：所见即所得的Markdown渲染，编辑同步预览
- **自动保存**：智能自动保存机制，告别意外丢失

### ⌨️ 高效快捷键体系
- `Ctrl+;` 快速插入日期
- `Ctrl+'` 快速插入时间  
- `Ctrl+S` 手动保存文档
- `Ctrl+C` 复制成功提示

### 🎨 精美视觉设计
- **响应式布局**：适配各种屏幕尺寸
- **主题系统**：支持多种主题切换
- **简约审美**：清爽界面，专注内容创作

### 🗺️ 知识图谱可视化
- **自动提取关联**：智能分析文档间关系
- **可视化展示**：直观呈现知识网络
- **快速跳转**：点击节点直接打开文档

## 🔧 可持续开发性

### 🏗️ 模块化架构
- **前后端分离**：RESTful API设计，便于多端扩展
- **组件化前端**：代码结构清晰，易于维护
- **插件化设计**：预留扩展接口，支持功能定制

### 💻 技术栈优势
- **Node.js服务端**：高性能异步处理，适合I/O密集型场景
- **原生前端技术**：零框架依赖，轻量高效
- **文件系统存储**：无需数据库，部署简单

### 📈 扩展能力
- **工作区管理API**：支持动态增删工作区
- **文档操作API**：完整的CRUD接口
- **搜索API**：支持关键词检索和图谱查询

### 🛡️ 稳定性保障
- **错误处理机制**：完善的异常捕获和友好提示
- **缓存策略**：智能缓存提升响应速度
- **日志系统**：分级日志便于问题排查
